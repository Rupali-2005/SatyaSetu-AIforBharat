# SatyaSetu

**Project:** SatyaSetu – Digital Reasoning Literacy Tool  
**Hackathon:** AWS AI for Bharat  

## Overview
SatyaSetu is a GenAI-powered tool designed to help communities detect logical fallacies and manipulation signals in articles and statements. It provides confidence scores, explanations, and balanced rewrites to promote critical thinking and informed understanding.

## Files in this Repo
- `requirements.md` – Detailed system requirements  
- `design.md` – Architecture and component design  
- `SatyaSetu-PPT.pptx` – Presentation slides for hackathon submission

## Note
Prototype focuses on English-language text input. The system aims to be educational, lightweight, and scalable.
